# Drum-kit
An interactive Drum Kit web application built using HTML, CSS, and JavaScript. Users can play drum sounds using keyboard keys or by clicking on buttons, with real-time audio feedback and visual effects.
---

## 🚀 Features

- 🖱️ Click-based drum interaction  
- ⌨️ Keyboard support (w, a, s, d, j, k, l)  
- 🔊 Realistic drum sounds  
- 🎨 Styled UI with background images  
- ⚡ Instant audio feedback  
- 💡 Simple and beginner-friendly project  

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (DOM Manipulation & Event Handling)**

---

## 🎮 Controls

| Key | Sound        |

| a   | Tom 2       |
| s   | Tom 3       |
| d   | Tom 4       |
| j   | Snare       |
| k   | Crash       |
